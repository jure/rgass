# Rgass.js

A JS implementation of RGASS, a CRDT synchronization algorithm presented in research paper by Xiao Lv et al: [http://dx.doi.org/10.1016/j.aei.2016.10.005](http://dx.doi.org/10.1016/j.aei.2016.10.005).

# To install

```
npm install
```

# To run tests

```
npm test
```

# To run battletest

```
npm run battletest
```
